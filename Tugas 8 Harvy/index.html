Apa itu Coding?

Coding adalah proses pengolahan code yang menggunakan bahasa pemograman, mulai dari menuliskannya, memperbaikinya, sehinggah memeliharanya. Kode yang dimaksud adalah perangkat pembangun suatu program komputer.

Coding adalah proses pengolahan code yang sementara itu, bahasa pemograman adalah suatu stkamur yang berfungsi memberikan perintah kepada komputer. Bahasa jenis terdiri dari suatu himpuan atau aturan sintaks dan semantic, sehinggah program-program komputer bisa terdefinisi. diantara bahasa pemograman yang paling populer di Indonesia di antaranya JAVA, PHP, C, Visual Basic, Python, C++, JavaScript, C#, Objective-c, ActionsScript.
